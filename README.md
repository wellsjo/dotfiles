## MacOS dotfiles

### Configs
- zsh
- vim
- tmux
- git

### Dependencies
- oh-my-zsh
- Homebrew
  - vim
  - tmux
  - icdiff
  - htop
  - go

### Install
Paste this in the terminal
```bash
git clone git@github.com:wellsjo/dotfiles.git ~/.dotfiles; ~/.dotfiles/setup
```

```
make sure /usr/local/bin comes before /usr/bin in $PATH
```
